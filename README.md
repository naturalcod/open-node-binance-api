# Binance Api Client
NodeJS Binance API Client
# Install
`npm i open-binance-api`

# Usage

```javascript
const OpenBinance = require("open-binance-api");

let binance_client = new OpenBinance(
  "API_KEY",
  "SECRET_KEY"
);

```
